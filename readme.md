# Python Mini Projects

A collection of beginner-friendly command-line Python projects.

- `project1.py` — Adventure game
- `project2.py` — Calculator
- `project3.py` — Contact manager
- `project4.py` — Tic-Tac-Toe
- `project5.py` — Quiz game

Run any project with:

```bash
python project1.py
```

Some projects use the included JSON files to store data and questions.
